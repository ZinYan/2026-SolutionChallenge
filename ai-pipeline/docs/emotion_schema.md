# 감정 분류 체계 초안 (Emotion Classification Schema)

본 문서는 AI 파이프라인에서 추출한 감정을 시각 및 촉각 피드백으로 연결하기 위한 기준 문서입니다. 이 문서는 팀원 간의 논의를 통해 지속적으로 업데이트될 예정입니다.

## 1. 8개 기본 감정 (8 Core Emotions)

기본적으로 Plutchik의 감정 수레바퀴(Wheel of Emotions)를 참고하여 8가지 주요 감정 + 1가지 중립 상태를 정의합니다.

1. **Joy (기쁨/즐거움)**: 긍정적이고 밝은 분위기 (예: 웃음, 환호)
2. **Sadness (슬픔)**: 우울하거나 침울한 분위기 (예: 우는 소리, 무거운 브금)
3. **Anger (분노/화남)**: 공격적이거나 격앙된 상태 (예: 소리지름, 빠른 비트)
4. **Fear (공포/두려움)**: 긴장감이나 위협을 느끼는 상태 (예: 미세한 노이즈, 날카로운 소리)
5. **Surprise (놀람/경악)**: 예상치 못한 갑작스러운 변화 (예: 점프스케어, 효과음)
6. **Disgust (혐오/불쾌)**: 거부감을 일으키는 상태 (예: 불협화음)
7. **Anticipation (기대/긴장)**: 무언가를 기다리며 서서히 고조되는 상태 (예: 빌드업 타악기)
8. **Trust (신뢰/평온)**: 안정적이고 평화로운 상태 (예: 잔잔한 앰비언스)
9. *Neutral (중립)*: 특별한 감정적 동요가 없는 일반적인 상태

## 2. 감정 강도 (Intensity)

각 감정은 `0.0` 부터 `1.0` 사이의 연속적인 값으로 강도를 가집니다.

* **0.0 ~ 0.3 (Low)**: 미세한 감정적 단서. 시각적/촉각적 피드백이 거의 없거나 매우 약함.
* **0.4 ~ 0.7 (Medium)**: 뚜렷하게 인지되는 감정. 표준적인 UI 효과 및 피드백.
* **0.8 ~ 1.0 (High)**: 극대화된 감정 상태. 강렬한 색상, 큰 떨림, 화면 이펙트 적용.

## 3. 감정 전이 (Shift)

이전 감정 상태에서 현재 감정 상태로 어떻게 변화했는지를 나타냅니다. (Layer 4 시각화 트랜지션에 사용)

* **Sudden (급격한 변화)**: 점프스케어나 반전처럼 순간적으로 감정이 바뀜. 시각 효과도 끊어지듯 즉각적으로 변환해야 함. (예: `Neutral` -> `Surprise`)
* **Gradual (서서히 변화)**: 빌드업되면서 감정이 자연스럽게 물들어가는 상태. 부드러운 색상/효과 페이드 인/아웃. (예: `Anticipation` -> `Fear`)
* **None (유지)**: 이전 상태와 동일하거나 미미한 변화만 있음.
